# Purple Times 

**Purple Times** é um site de notícias e entretenimento, com artigos, galeria de vinis, seções de notícias, previsão do tempo e formulário de contato. O projeto foi desenvolvido usando **HTML e CSS puro**, com foco em responsividade e efeitos visuais modernos.

---

## Estrutura do Projeto

/site-main
│
├── index.html # Página inicial
├── noticia.html # Página de artigo
├── contato.html # Página de contato
├── entretenimento.html # Página de entrenimento
├── style.css # Estilo principal
├── entretenimento.css # Estilo do entretenimento
├── noticia.css # Estilo da noticia
├── contato.css # Estilo do contato
└── img/ # Imagens do site


---

##  Tecnologias Usadas
- HTML5
- CSS3 ()
- Fontes do Google Fonts: Merriweather e Times New Roman
- Imagens otimizadas para web

---

##  Responsividade
O site foi desenvolvido para funcionar bem em:
- Desktop (maiores de 1024px)
- Mobile (até 768px)

Elementos responsivos incluem:
- Galeria de vinis com scroll horizontal em celulares
- Artigos, cards e prateleiras ajustáveis
- Header, rodapé e botão de contato fixo

---

##  Funcionalidades
- **Header** fixo e estilizado
- **Menu principal** com links e hover
- **Seção de notícias** com destaque e cards
- **Galeria de vinis** com efeito 3D e hover
- **Formulário de contato** com validação visual
- **Rodapé** com informações e links sociais
- **Botão de contato** fixo na tela
- **Seção de comentários** em artigos
